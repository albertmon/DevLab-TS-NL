
Command-line (without IDE)

1) download grails
2) install grails (unzip it to a place)

3) set GRAILS_HOME to where grails is installed (system variables)

4) windows console:
    - cd into grails-app
    - type 'grails', this should start the interactive grails-mode
    - type 'run-app'
    - browse to the url you get in the output
    - smile ;-)
