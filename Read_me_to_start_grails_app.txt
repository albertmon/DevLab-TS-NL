
Command-line (without IDE)

1) download grails 2.2.4 (http://grails.org/2.2.4%20Release%20Notes)

2) install grails (unzip it to a place)

3) 
    - set GRAILS_HOME to where grails is installed (system variables) (GRAILS_HOME C:\stefan\apps\grails-2.2.4)
    - add GRAILS_HOME to your Path variable (...;%GRAILS_HOME%\bin;...)

4) windows console:
    - cd into grails-app
    - type 'grails', this should start the interactive grails-mode
    - type 'run-app'
    - browse to the url you get in the output
    - smile ;-)
